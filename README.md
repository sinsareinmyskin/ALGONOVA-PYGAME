# ALGONOVA-PYGAME
This game is made using VSCODE Algonova.

This is a simple game about a labyrinth.

Features
- Simple graphics
- Intuitive controls
- Enemies
- A game

How to download
Clone repository ini:
git clone https://github.com/.snH/ALGONOVA-PYGAME.git
cd ALGONOVA-PLAYGAME

Install dependencies:
pip install pygame

Jalankan game:
python main.py
